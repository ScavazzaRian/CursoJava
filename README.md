# Curso de Java Completo - Nélio Alves

Este repositório contém os exercícios e projetos desenvolvidos durante o "Curso mais didático e completo de Java e OO, UML, JDBC, JavaFX, Spring Boot, JPA, Hibernate, MySQL, MongoDB e muito mais". O foco principal é o domínio da linguagem Java, Programação Orientada a Objetos e boas práticas de arquitetura.

## Tecnologias Utilizadas

* Java (JDK 8 ou superior)
* IntelliJ IDEA / Eclipse
* Git e GitHub

## Conteúdo Estudado

Os códigos estão organizados por tópicos, abrangendo:

* Lógica de Programação e Sintaxe Básica
* Programação Orientada a Objetos (Classes, Atributos, Métodos)
* Construtores, Sobrecarga, Encapsulamento
* Comportamento de Memória, Arrays e Listas
* Herança, Polimorfismo e Classes Abstratas
* Tratamento de Exceções
* Trabalhando com Arquivos
* Interfaces (Injeção de Dependência e Inversão de Controle)
* Generics, Set e Map
* Programação Funcional (Lambda)
